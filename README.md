# 🩺 MediVision AI — Medical Image Analysis Platform

> An AI-powered medical image analysis web app built with Streamlit. Upload any medical image (X-Ray, MRI, CT, Ultrasound) and get a detailed visual report generated by a Vision AI — not pixel statistics. Includes a multilingual medical chatbot, doctor finder, and full analysis history.

---

## ✨ Features

| Feature | Description |
|---|---|
| 🔬 **Vision AI Report** | Sends the actual image to Groq Vision (Llama 4) or Gemini Vision — describes what it visually sees |
| 💬 **Medical Chatbot** | Answers questions about symptoms, treatments, and conditions with real educational information |
| 🏥 **Doctor Finder** | Enter your city → get Google Maps links for nearby hospitals, radiologists, and specialists |
| 📋 **Analysis History** | Every report is saved locally — search, review, and reload past analyses into the chatbot |
| 🌐 **Multilanguage** | 11 languages: English, Tamil, Hindi, Telugu, Malayalam, Kannada, French, Spanish, Arabic, German, Chinese |
| 🎨 **Segmentation Overlay** | Visual mask overlay on the image with adjustable color and opacity |

## 🚀 Getting Started

### Prerequisites
- Python 3.10 or higher
- A free API key from [console.groq.com](https://console.groq.com) (recommended) or [aistudio.google.com](https://aistudio.google.com)

### 1. Clone the Repository

```bash
git clone https://github.com/hemanth_kumar02/medivision-ai.git
cd medivision-ai
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Set Up API Keys

Create a `.env` file in the root directory:

```env
# Groq — FREE, 14,400 requests/day, supports Llama 4 Vision
# Get your key at: https://console.groq.com
GROQ_API_KEY=your_groq_api_key_here

# Gemini — FREE, 1,000 requests/day, supports gemini-1.5-flash Vision
# Get your key at: https://aistudio.google.com
# Optional — used as fallback if Groq is unavailable
GEMINI_API_KEY=your_gemini_api_key_here
```

### 4. Run the App

```bash
streamlit run app.py
```

Open your browser at `http://localhost:8501`

## 🤖 AI Vision Pipeline

```
Image Upload
     │
     ▼
Groq Vision (Llama 4 Scout/Maverick)  ──► Report ✅
     │ (if unavailable)
     ▼
Gemini Vision (gemini-1.5-flash)       ──► Report ✅
     │ (if unavailable)
     ▼
Informative fallback message           ──► Setup guide shown
```

The AI **looks at the actual image** and describes:
1. Image type & view (X-ray PA, MRI axial, CT coronal, etc.)
2. Visible anatomical structures
3. Key visual findings
4. Impression & possible conditions (educational)
5. Suggested next steps

## 🔒 Privacy & Safety

- Images are sent to the AI API only for analysis and are not stored by this app
- Analysis history is saved **locally** on your machine only (`medivision_history.json`)
- The `.env` file containing API keys is excluded from Git via `.gitignore`
